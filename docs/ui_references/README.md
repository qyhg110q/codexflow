# UI References

`docs/ui_references/` stores visual reference material for CodexFlow clients.
This includes HTML prototypes, screenshots, and notes that guide UI implementation.

## Current References

- `android_mobile/`
  - Android / mobile-first HTML prototype for CodexFlow session control

## Conventions

- HTML prototypes should use `*_prototype.html`.
- Screenshots should use `*_reference.png` or a similarly explicit surface name.
- Each reference directory should include a short `README.md` explaining preview steps and implementation mapping.
